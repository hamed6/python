# python
python GUI project - 2.7 py , Tkinter

The app is about to save note/task(technically any short text) plus the date.
Tkinter library has used for this app which has limited number of widgets.

Some tips:

- There is no text validation behind this app , so any string even empty one is accepted.
- Comma has used as seperator , so dont use in your text.
- Follow one either of this way to save the date as there is no date validation. 1.2.2017 or 01.02.2017 , otherwise, it will be processed
  as 2 different date.
